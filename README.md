School Admin is a web-based application for managing students, teachers, and classes efficiently.
It allows administrators to add, edit, and track student information, attendance, and results from a single dashboard.

Features
# Add, update, and delete student and teacher records
# Manage attendance and subjects
# Generate progress reports
# Role-based access for admin, teacher, and student
# Backend: PostgreSQL, Node.js (Express)
# Frontnd: React.js
